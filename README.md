# Game of Life

Two independent Game of Life implementations.

- **GameOfLife.py** — classic Conway's Game of Life (pygame).
- **intetactive_gol.py** — SmoothLife, a continuous generalization of GoL (PySide6 UI with parameter sliders).

## Run

```sh
uv sync
uv run python GameOfLife.py
uv run python intetactive_gol.py
```
